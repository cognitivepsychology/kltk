 13  USAGE 
 14  ===== 
 15   
 16  You have a Sejong parsed corpus file named 'BGJO0150.bnk'.  
 17  At first, remove XML tags and check file encoding. 
 18  :: 
 19   
 20          file = codecs.open('BGJO0150-noxml.bnk' encoding='utf-8') 
 21          fw = ForestWalker(file) 
 22   
 23  L{ForestWalker} is an iterator of forest (treebank). It isn't the 
 24  treebank itself. It does NOT load trees to the memory. It's 
 25  light and fast! 
 26  :: 
 27   
 28          for tree in fw: 
 29                  do (tree) 
 30   
 31  If you want to get a fully-loaded L{TreeBank} object, try getTreeBank(). 
 32  It takes some time according to the file size.  
 33  :: 
 34   
 35          treebank = fw.getTreeBank()
