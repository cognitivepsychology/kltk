path_to_terminal

get a tree source line and return path_to_terminal. path_to_terminal corresponds to one line of TreeBank file. It is a tuple of nodes, a terminal node and the number of parentheses at the end of the source line. for example,

source line :

       (S      (NP_AJT         (DP 그/MM)

path_to_terminal :

       ["S", "NP_AJT", "DP", "그/MM", 1]
